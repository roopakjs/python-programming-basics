# Python Programming Basics

This repository contains basic Python programs created for learning core programming concepts and improving logical thinking.

## Programs Included
- hello_world.py – Prints a simple message
- calculator.py – Performs basic arithmetic operations
- palindrome.py – Checks whether a number is a palindrome
- prime_number.py – Checks whether a number is prime
- fibonacci.py – Generates Fibonacci series
- number_guessing_game.py – Simple number guessing game
- file_handling.py – Demonstrates file read and write operations
- password_strength_checker.py – Checks password strength

## Concepts Covered
- Input and output
- Conditional statements
- Loops
- Functions and logic building
- String manipulation
- File handling

## Technologies Used
- Python

## Purpose
This repository is intended for beginners who are learning Python and want to strengthen their programming fundamentals.
